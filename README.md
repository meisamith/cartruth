# CarTruth 🚗

India's honest car ownership guide. Not the showroom pitch — real mileage, real service costs, real problems.

## Live App
https://web-production-dfc9.up.railway.app

## Features
- Car Report — real mileage, true ownership cost, common problems
- Compare Mode — side by side comparison with personalised winner
- Used Car Wizard — 10 question inspection + price verdict
- Budget Calculator — top 3 cars for your budget and city
- Fair Price Checker — is this used car priced fairly?
- PWA — installable on phone home screen

## Tech Stack
- Python + Flask
- Claude API (Haiku model)
- PostgreSQL (Railway)
- Vanilla JS + CSS
- Deployed on Railway
